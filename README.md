# Titanic Data Analysis

## Overview
This project involves an analysis of the Titanic dataset, focusing on understanding the factors that contributed to passenger survival during the ill-fated maiden voyage. Through data visualization and statistical analysis, insights into the socio-economic demographics of the passengers and their likelihood of survival are explored.

## Features
- Data Cleaning and Preprocessing
- Exploratory Data Analysis (EDA)
- Data Visualization with Matplotlib and Seaborn
- Machine Learning Model Implementation
- Interactive Dashboards

## Tech Stack
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Flask for web application

## Installation
To install the required dependencies, run the following command:
```bash
pip install -r requirements.txt
```

## Usage
1. Clone the repository:
   ```bash
   git clone https://github.com/MurtazaMajid/Titanic-Data-Analysis.git
   cd Titanic-Data-Analysis
   ```
2. Run the main script:
   ```bash
   python main.py
   ```
